# Codex Forms

An Angular application scaffolded for building dynamic forms using standalone APIs, strict type checking, Tailwind CSS styling, and client-side routing between builder and preview experiences.

## Development

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm start
   ```

3. Build for production:

   ```bash
   npm run build
   ```
